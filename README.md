# KAKAO-CLONE

html, css, fontawesome을 사용하여 카카오톡의 화면단을 구현 했습니다.


- chat.html -> 대화방 화면 
- chats.html -> 채팅 목록 화면
- find.html -> 검색 화면
- friends.html -> 친구 목록
- index.html -> 로그인 화면, 기능은 X
- more.html -> 기타 화면
- settings.html -> 환경 설정 화면
